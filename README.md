# Student Chat App

## Опис проєкту
Student Chat App — це чат-додаток для студентів, який дозволяє швидко та зручно спілкуватися між собою в межах навчальних груп, курсів або предметів.

## Яку проблему вирішує
Додаток спрощує комунікацію між студентами та допомагає:
- обмінюватися повідомленнями;
- швидко передавати інформацію про пари, дедлайни та події;
- підтримувати зв’язок у групі без використання сторонніх месенджерів.

## Основні функції
- Реєстрація та вхід користувачів
- Обмін текстовими повідомленнями
- Створення групових чатів
- Надсилання файлів
- Перегляд історії повідомлень

## Цільова аудиторія
- Студенти коледжів та університетів
- Старости груп
- Навчальні команди
