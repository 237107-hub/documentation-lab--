# Керівництво користувача

## Початок роботи
1. Відкрийте вебсайт або мобільний застосунок Student Chat App.
2. Зареєструйтесь, вказавши ім’я та email.
3. Увійдіть у систему.
4. Оберіть або створіть чат своєї групи.

## Робота з чатом
- Для надсилання повідомлення введіть текст у поле вводу та натисніть кнопку “Надіслати”.
- Для створення групового чату натисніть “Створити групу”.
- Можна надсилати файли та зображення.

## FAQ (Питання та відповіді)

**П: Чи можна змінити ім’я користувача?**  
В: Так, у налаштуваннях профілю.

**П: Чи можна користуватися додатком безкоштовно?**  
В: Так, додаток повністю безкоштовний.

**П: Чи зберігається історія повідомлень?**  
В: Так, всі повідомлення зберігаються в чаті.

**П: Чи можна створювати кілька чатів?**  
В: Так, користувач може створювати необмежену кількість чатів.
