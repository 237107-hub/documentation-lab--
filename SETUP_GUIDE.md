# Інструкція з встановлення та запуску

## Системні вимоги
- Комп’ютер або смартфон
- Сучасний веббраузер (Chrome, Firefox, Edge)
- Доступ до Інтернету

## Кроки запуску
1. Відкрити браузер.
2. Перейти на сайт Student Chat App.
3. Зареєструватися або увійти в акаунт.
4. Обрати або створити чат.

## Додаткові інструменти
- Email для реєстрації
- Стабільне інтернет-з’єднання

## Перевірка роботи
Якщо після входу користувач може:
- бачити список чатів;
- надсилати повідомлення;
- отримувати відповіді  
— значить система працює коректно.
